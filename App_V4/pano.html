ملاحظات مهمة

حتى لا نعيد العمل عند الانتقال إلى Supervisely GPU.

  1. ثبات عقد البيانات (API Contract): سنثبت شكل الاستجابة داخل التطبيق. عند تغيير مزود الـAI (Colab → Supervisely GPU) لن تغيّر الواجهة، فقط الـURL/التوثيق.
  2. Auto API Mode: التطبيق سيجرّب /infer (Multipart) ثم fallback إلى /analyze. عند تجهيز Supervisely سنجعل السيرفر يعيد نفس الـJSON.
  3. PWA + Mobile: الرفع والمعالجة تتم مع مؤشرات تحميل واضحة وتخزين إعداد الرابط محليًا.
  4. التقرير الطبي النصّي: سيتم تحسينه تلقائيًا عبر RRZ AI Gateway (Netlify + Gemma). إذا تعذر الاتصال، سيبقى التقرير المحلي كبديل.
